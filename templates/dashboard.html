Platform Setup Guides
iOS (iPhone / iPad)
Install MDM Profile or configure manually via Wi-Fi settings.
  1. Download the DNS profile from this dashboard
  2. Go to Settings → General → VPN & Device Management
  3. Tap the installed profile and confirm
  4. For enforcement: Set a ScreenTime passcode, then go to ScreenTime → Content & Privacy Restrictions → Profile Changes → Don't Allow
Android
Use Private DNS (DoT) for system-wide protection.
  1. Go to Settings → Network & Internet → Advanced → Private DNS
  2. Select Private DNS provider hostname
  3. Enter: dns-pc.example.com
  4. Save — all DNS traffic is now filtered
macOS
Configure DNS in System Preferences.
  1. Open System Preferences → Network
  2. Select your active connection and click Advanced
  3. Go to the DNS tab
  4. Click + and add 127.0.0.1
  5. Remove any other DNS servers, click OK and Apply
Windows
Set DNS server in network settings.
  1. Go to Settings → Network & Internet → Wi-Fi/Ethernet
  2. Click DNS server assignment → Edit
  3. Switch to Manual, enable IPv4
  4. Enter 127.0.0.1 and save
Router
Protect all devices on your network at once.
  1. Log into your router admin panel (typically 192.168.1.1)
  2. Find WAN Settings or DHCP Server Settings
  3. Change DNS servers to 127.0.0.1 (or the server's LAN IP)
  4. Save and reboot the router
